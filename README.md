# Covid-19

Criado com objetivo de conhecimento com as tecnologias `Typescript` e `ContextApi` do ReactJs.

Funcionamento é exibir os dados de covid-19 de cada país que é selecionado no select que está no `Header` da aplicação.

# Getting Started with Create React App

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Available Scripts
